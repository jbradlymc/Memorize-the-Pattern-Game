HOW TO PLAY GAME:

STEP 1: using button5 and button6, the player can increment or decrement the number of patterns to be generated respectively (Maximum of 9)
STEP 2: press button4 so that the RGB LED will start to generate a random pattern
STEP 3: repeat the same pattern shown by the RGB LED using buttons 1, 2, and 3 to light LEDS red, green, and blue respectively

NOTE: your remaining life/lives is shown in the 7-segment display, wherein each player has maximum of 3 lives in each game. Once your life has run out, the game will reset.
